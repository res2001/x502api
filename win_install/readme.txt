Данная установка содержит файлы для написания пользовательских
программ (как 32-х, так 64-х битных), для работы с крейтовой системой LTR.


Содержимое директорий:
bin       
    /x86            - 32-битные версии библиотек (dll) (также устанавливаются в системную директорию). 
    /x64            - 64-битные версии библиотек (dll) (также устанавливаются в системную директорию). 	
    /ltrModulesNet  - Обертка .Net для написания программ на C# или LabView
    /firmware       - Файлы прошивок для контроллеров/ПЛИС модулей LTR,		 
include             - Содержит заголовочные файлы, необходимые для разработки программ на C/C++
    /pascal         - Старая версия файлов для написания программ на Delphi/Pascal (Delphi7 и ниже)
                      (оставлена для совместимости со старыми проектами)	 
    /pascal2        - Новая версия файлов для написания программ на Delphi/Pascal (поддержка и Delphi 7 и XE)
lib                 - Содержит .lib файлы для различных компиляторов, необходимыхе для линковки с библиотекой при написании
		         программ на C/C++. Эти файлы должны быть включены в проект или указаны в настройках 
                         среды для проекта.



Изменение расположения файлов по сравнению с версией 1.27.3 и ниже:
    - dll файлы перенесены в bin/x86 из lib
    - файлы прошивок перенесены в bin/firmware из lib
    - ltrModuleNet перенесена в bin/ltrModulesNet из  lib/Net
