Данный пример демонстрирует работу с модулями L502 и E502 из среды Visual Basic 6.

Модули x502api.bas, e502api.bas и l502api.bas включают необходимые объявления типов
и функций из библиотеки x502api и могут использоваться без изменений в пользовательском проекте
(об отличиях в описании функций написано в руководстве программиста).

Пример по нажатию кнопки устанавливает соединение с выбранным устройством, настраивает его (параметры задаются в коде),
выполняет сбор блока данных с расчетом среднего и закрывает соединение. 
Для простоты все операции выполняются в обработчике кнопки в основном потоке (т.к. данный пример предназначен
в первую очередь для демонстрации работы именно с самим модулем, а не работу с потоками и другими средствами).
