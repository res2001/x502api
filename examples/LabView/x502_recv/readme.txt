Пример демонстрирует вариант написания программы потокового сбора данных с АЦП и DIN на LabView для модулей L502 или E502.
В примере сперва обнаруживаются все модули, подключенные по интерфейсам PCIe и USB и пользователю предоставляется список для выбора модуля.
Также есть опция использовать устройство подключенное по Ethernet с заданным IP-адресом.
По нажатию кнопки СТАРТ устанавливаться связь с выбранным устройством и идет сбор данных.

Пример использует библиотеку lpcieNet.dll, которая должна распространяться с примером (или устанавливаться в GAC), 
которая требует наличия в системе билиотек x502api.dll, l502api.dll и e502api.dll (которая ставится в систему вместе с SDK)

Пример сохранен в для LabView 8 и LabView 2010 (так как новые версии LabView могут не совсем корректно открывать проекты от LabView 8)
